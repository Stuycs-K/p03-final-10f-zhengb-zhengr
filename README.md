### Project Description:

Our project is an online chatroom where many people can connect and chat with eachother. This includes, the usser's name, message and time they sent it. On the left, you will see a server list. This will update based on who joins and leaves. Additionally a join and leave message will pop up in the chat.


### Instructions:
User needs to oepn at least 3 terminals. First, run make compile, then run make server on one, then make client on the others. This will connect the user to the chatroom. User will now be able to change their username and type. To leave, press CTRL + c.
